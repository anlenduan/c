## 全局快捷键

```c
zz: 屏幕中心光标
Ctrl + e: 将屏幕向下移动一行（不移动光标）
Ctrl + y: 将屏幕向上移动一行（不移动光标）
Ctrl + b: 将屏幕向上移动一页（光标到最后一行）
zb: 将光标置于屏幕底部
gk: 向上移动光标（多行文本）
k: 向上移动光标
{: 跳转到上一段（或编辑代码时的功能/块）
;: 重复上一个 f、t、F 或 T 动作
Ctrl + f: 将屏幕向下移动一页（光标到第一行）
Ctrl + d: 将光标和屏幕向下移动 1/2 页
Ctrl + u: 将光标和屏幕向上移动 1/2 页
i: 在光标前插入
I: 插入到行首
a: 在光标后插入（追加）
A: 在行尾插入（追加）
o: 在当前行下方添加（打开）一个新行
O: 在当前行上方添加（打开）一个新行
ea: 在单词末尾插入（附加）
Ctrl + h: 在插入模式下删除光标前的字符
Ctrl + w: 在插入模式下删除光标前的单词
Ctrl + j: 在插入模式下在光标位置添加换行符
Ctrl + t: 插入模式下缩进（向右移动）一行 shiftwidth
Ctrl + d: 在插入模式下取消缩进（向左移动）行一个 shiftwidth
Ctrl + n: 在插入模式下，在光标前插入（自动完成）下一个匹配项
Ctrl + p: 在插入模式下，在光标前插入（自动完成）上一个匹配项
Ctrl + rx: 插入寄存器 x 的内容
Ctrl + ox: 暂时进入正常模式发出一个正常模式命令x。
Esc + Ctrl + c: 退出插入模式
$: 跳至行尾
:h[elp] keyword: 打开关键字帮助
:ter[minal]: 打开终端窗口
K: 打开光标下的单词的手册页
h: 向左移动光标
j: 向下移动光标
:clo[se]: 关闭当前窗格
:sav[eas] file: 将文件另存为
l: 向右移动光标
gj: 向下移动光标（多行文本）
M: 移至屏幕中间
L: 移至屏幕底部
w: 跳转到单词的开头
W: 跳转到单词的开头（单词可以包含标点符号）
e: 跳转到单词末尾
b: 跳转到单词的开头
B: 向后跳转到单词的开头（单词可以包含标点符号）
ge: 跳转到单词末尾
gE: 向后跳转到单词的末尾（单词可以包含标点符号）
%: 将光标移动到匹配的字符（默认支持对：“()”、“{}”、“[ ]”）
0: 跳转到行首
^: 跳转到该行的第一个非空白字符
g_: 跳转到该行的最后一个非空白字符
G: 转到文档的最后一行
gd: 移至本地声明
gD: 移至全局声明
fx: 跳转到下一个出现的字符 x
tx: 跳转到下一个字符 x 出现的位置
Fx: 跳转到字符 x 的上一次出现位置
Tx: 跳转到上一次出现字符 x 之后
,: 向后重复上一个 f、t、F 或 T 动作
}: 跳转到下一段（或编辑代码时的功能/块）
zt: 将光标置于屏幕顶部
5gg 5G: 转到第 5 行
gg: 转到文档的第一行
H: 移至屏幕顶部
E: 跳转到单词的末尾（单词可以包含标点符号）
```
